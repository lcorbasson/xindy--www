I have further improved the installation step that now uses a
reasonably portable install mechanism. Therefore the Makefiles have
changed slightly. The path to Perl is now configurable and is set to
/usr/bin/perl on Linux, and /usr/local/bin/perl on the other platform
as the default value.

Additionally the TeX-modules are rearranged as disscussed yesterday.
